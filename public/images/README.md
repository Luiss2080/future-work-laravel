## Recomendaciones de Imágenes

### Tamaños Recomendados:
- Logo: 200x60px (PNG con transparencia)
- Hero: 1920x1080px (JPG optimizada)
- Profesionales: 400x300px (JPG optimizada)
- Testimoniales: 80x80px (JPG circular)

### Optimización:
- Usar WebP cuando sea posible
- Comprimir JPG al 85% de calidad
- PNG solo para transparencias
- Lazy loading para imágenes below the fold

### SEO:
- Alt text descriptivo en español
- Nombres de archivo descriptivos
- Dimensiones apropiadas para responsive
